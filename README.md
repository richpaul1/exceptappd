exceptappd
==========

Exception Tracker

Installation :

Required Frameworks :
1. node.js
2. bower

Installation :

1. Clone from repository : 
>git clone https://github.com/richpaul1/exceptappd.git

2. Change into the direcotry
> cd exceptappd

3. Download the dependencies
> bower install

4. start node.js
> npm start

5. Open browser to :
http://localhost:3000
